# NeuralForge PS3.23 – AI Emergency Triage System (SDG 3)

## Problem
During disasters and emergencies, chaos, unstructured information, and poor coordination delay rescue operations and cost lives.

## Solution
NeuralForge PS3.23 is an AI-powered emergency triage and coordination system that:
- Accepts emergency messages
- Uses AI to classify severity
- Assigns priority
- Routes requests to appropriate rescue teams
- Creates structured, actionable rescue decisions

## Core Features
- AI emergency classification
- Severity scoring
- Priority assignment
- Rescue team routing
- Real-time decision output
- Emergency coordination logic

## System Flow
User → AI Analysis → Severity Engine → Priority Engine → Routing Engine → Rescue Assignment

## Tech Stack
- Backend: FastAPI (Python)
- AI Logic: Rule-based AI classification
- Frontend: HTML + JS Demo UI
- Architecture: Modular microservices structure

## SDG Alignment
SDG 3 – Good Health & Well-Being  
Improves emergency response coordination and disaster triage.

## Hackathon Vision
NeuralForge PS3.23 demonstrates how AI can transform chaos into structured emergency action in disaster and crisis situations.
